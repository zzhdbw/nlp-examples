# nlp-examples

使用torch和transformers库以实现nlp中不同任务的代码

使用BERT进行情感分析：[sentiment_classification.py](https://github.com/zzhdbw/nlp-examples/blob/main/sentiment_classification.py)

使用fasttext库提供的word2vec词向量进行情感分析：[sentiment_classification_w2v.py](https://github.com/zzhdbw/nlp-examples/blob/main/sentiment_classification_w2v.py)

使用fasttext库提供的word2vec词向量+GCN进行情感分析：[sentiment_classification_w2v_gcn.py)](https://github.com/zzhdbw/nlp-examples/blob/main/sentiment_classification_w2v_gcn.py)

使用BERT进行命名实体识别任务：[sequence_labeling_ner.py](https://github.com/zzhdbw/nlp-examples/blob/main/sequence_labeling_ner.py)

使用BERT+CRF进行命名实体识别任务：[sequence_labeling_ner_crf.py](https://github.com/zzhdbw/nlp-examples/blob/main/sequence_labeling_ner_crf.py)

使用BERT+TPLinker进行联合关系抽取任务：[relation_extraction_joint_TPLinker.py ](https://github.com/zzhdbw/nlp-examples/blob/main/relation_extraction_joint_TPLinker.py)#TODO
